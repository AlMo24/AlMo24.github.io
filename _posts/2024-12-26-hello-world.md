# Hello World

Ya. Akhirnya saya belajar bikin blog lagi setelah 20 tahun lebih vakum dari web developing.
Hari ini saya belajar buanyak hal lagi. Baiklah, demikian "pembukaan" web ini, nanti kita lanjut lagi.
